## Getting Started

To push this repository to GitHub:

1. Initialize Git:
   ```bash
   git init
   ```

2. Stage files:
   ```bash
   git add .
   ```

3. Commit changes:
   ```bash
   git commit -m "Initial commit: Adding landing page template"
   ```

4. Add remote:
   ```bash
   git remote add origin <your-repository-url>
   ```

5. Push to GitHub:
   ```bash
   git push -u origin main
   ```
